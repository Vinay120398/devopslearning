# shellscript

**steps**
* Create a repo in Github
* Clone repo in a local laptop
* Start Developing
* Add files to temp area
```
git add <file-name>
```
* Commit changes
```
git commit -m "why-you-commited-this"
```
* Push the chnages to Github

```
git push -u orign main
```
**naming convention : use small case with hypones"-" and no spaces**
